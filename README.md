# KEC - College Website

A modern, responsive, and professional College Website for KEC built using HTML, CSS, and JavaScript with a white, black, and orange theme.

## Features

- **Home Page** - Hero slider with KEC branding, admission banner, apply now button
- **Navigation** - Sticky nav, dropdown menus, mobile hamburger menu, smooth scrolling
- **About** - History, vision, mission, principal's message, chairman's message, accreditations
- **Departments** - 11 department cards (ECE, EEE, CSE, CSE-AI&ML, CSE-DS, AI, Mechanical, Humanities & Sciences, Civil, MBA, R&D) with search
- **Courses** - UG/PG tabs with duration, eligibility, and intake details
- **Admissions** - Process, eligibility, documents, fee structure, scholarships
- **Faculty** - 9 faculty cards with search by name or department
- **Campus Facilities** - 10 facility cards (library, labs, hostel, sports, etc.)
- **Placements** - Statistics, top recruiters, internship opportunities
- **Events** - Countdown timer, 6 event cards with register buttons
- **Gallery** - Real college event/graduation photos from Unsplash with lightbox preview
- **Latest News** - 6 news cards with dates
- **Achievements** - 6 achievement cards
- **Student Testimonials** - Auto-sliding testimonials
- **FAQ** - Accordion-style questions
- **Contact** - Address, phone, email, map, validated contact form
- **Footer** - Quick links, social media, copyright

## How to Run

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. No server or build tools required
