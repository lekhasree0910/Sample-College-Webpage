# PastelBeats - Music Player

A modern, responsive music player built with HTML, CSS, and JavaScript featuring a pastel color palette.

## Features

- **Playback Controls**: Play, pause, previous, next, restart, mute/unmute
- **Progress Bar**: Click/drag to seek, live time display
- **Volume Controls**: Slider with mute toggle, persisted via Local Storage
- **Playlist**: Scrollable with current song highlight, auto-play next
- **Favorites**: Add/remove with heart animation, stored in Local Storage
- **Search**: Real-time filtering by title, artist, album
- **Album Section**: Browse by album with cover art
- **Categories**: Filter by Pop, Rock, Hip-Hop, Classical, Lo-Fi, Jazz, EDM
- **Recently Played**: Track history with timestamps
- **Lyrics Panel**: Toggle lyrics display while playing
- **Equalizer**: Animated bars synced to playback
- **Theme Switcher**: Light/Dark mode with Local Storage persistence
- **Keyboard Shortcuts**: Space (play/pause), arrows (prev/next), M (mute)
- **Toast Notifications**: Non-intrusive feedback
- **Loading States**: Spinner while audio loads
- **Responsive Design**: Desktop, tablet, mobile layouts
- **Scroll Animations**: Fade-in effects on section changes

## Setup

1. Add your `.mp3` files to the `music/` folder
2. Add cover images to the `images/` folder
3. Update song data in `script.js` to match your files
4. Open `index.html` in a browser or serve with a local server

## Folder Structure

```
Music-Player/
  index.html
  style.css
  script.js
  music/      -- Add .mp3 files here
  images/     -- Add cover art here
  lyrics/     -- Add .txt lyric files here
  README.md
```

## Color Palette

Pastel pink, lavender, mint, peach, baby blue, and cream tones with smooth gradients.
